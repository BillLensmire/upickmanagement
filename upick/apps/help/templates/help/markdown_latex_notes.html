Markdown and LaTeX Notes

About LaTeX Notes

LaTeX syntax is different from Markdown. The issue that LaTeX uses the $ symbol for math equations, which can conflict with Markdown formatting. If you want to mention dollar amounts in your notes, for example 'the seed cost $5.60', leave the checkbox for LaTeX unchecked.

If you are just entering a simple note without any math equations, you can leave the checkbox for LaTeX unchecked.

If you need to enter equations, then put that into a separate note and check the LaTeX checkbox.
